Data Types

String
text data enclosed with quotes. let myString = 'My String'
Number
Number data that can be modified with primitive operators.

Variables

var
Function-scope variable that can be accessed throughout a function. var myVariable = 'Function-scope'
let
Block-scoped variable that can be accessed within the {} but not outside of it. Let signals that the value can be changed. let myVariable = 'block-scope'
const
Constant variable. The value cannot be reassigned. Constant variables must be assigned a value when the variable is declared const myVariable = 'Cannot be changed'

Operators

Primitive operators
Basic math operators. Addition = +, Subtraction = -, multiplication = *, division = /
Mathematical assignment operators
Combines a primitive operator with the value assignment operator '=" let x = 20; x -= 5; // Can be written as x = x - 5 console.log(x); // Output: 15 let y = 50; y *= 2; // Can be written as y = y * 2 console.log(y); // Output: 100
The Increment and Decrement Operator
The increment operator will increase the value of the variable by 1. The decrement operator will decrease the value of the variable by 1. let a = 10; a++; console.log(a); // Output: 11 let b = 20; b--; console.log(b); // Output: 19
